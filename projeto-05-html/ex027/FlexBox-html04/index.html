O que é FlexBox ?

O Flexbox (Flexible Box Module) é um modelo de layout CSS3 unidimensional projetado para organizar, alinhar e distribuir espaço entre itens dentro de um contêiner, mesmo quando seu tamanho é desconhecido ou dinâmico.

Caracteristicas do FlexBox:

  • Organização Flexível dos Elementos
  • Trabalha com Eixos
  • Alinhamento Simplificado
  • Layout Responsivo
  • Controle de Direção
  • Distribuição Automática de Espaço

O Flexbox é um modelo de layout do CSS criado para facilitar a organização e o alinhamento de elementos em páginas web. Sua principal característica é a flexibilidade, permitindo que os itens cresçam, diminuam e se adaptem automaticamente ao espaço disponível.
