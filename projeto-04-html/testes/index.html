O que é HTML e CSS

HTML e CSS são duas tecnologias básicas usadas para criar páginas na web.

Home Curiosidades Contato

O que é HTML ?

É a linguagem que define a estrutura de uma página. Com HTML você organiza o conteúdo, como títulos, textos, imagens, links, tabelas etc.

O que é CSS ?

É a linguagem usada para definir o estilo visual da página. Com CSS você muda cores, tamanhos, fontes, espaçamentos, layouts e muito mais.
